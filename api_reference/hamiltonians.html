class qvarnet.hamiltonians.HarmonicOscillator(model: Module, omega: float = 1.0)[source]

Bases: GeneralHamiltonian

Harmonic oscillator Hamiltonian class

This class implements the Hamiltonian for a 1D, single particle quantum harmonic oscillator.

The Hamiltonian is given by:

\[H = -\frac{\hbar^2}{2m} \frac{d^2}{dx^2} + \frac{1}{2} m \omega^2 x^2\]

where \(\hbar\) is the reduced Planck’s constant, \(m\) is the mass of the particle, and \(\omega\) is the angular frequency of the oscillator. The kinetic energy operator is given by: \(T = -\frac{\hbar^2}{2m} \frac{d^2}{dx^2}\) The potential energy operator is given by: \(V = \frac{1}{2} m \omega^2 x^2\) The interaction potential is set to zero for this case.

Parameters:

omega (float) – Angular frequency of the oscillator. Default is 1.0.

class qvarnet.hamiltonians.GrossPitaevskii(model: Module, omega: float = 1.0, g: float = 1.0)[source]

Bases: GeneralHamiltonian

Gross-Pitaevskii Hamiltonian class

This class implements the Hamiltonian for a 1D, single particle Gross-Pitaevskii equation.

The Hamiltonian is given by:

\[H = -\frac{\hbar^2}{2m} \frac{d^2}{dx^2} + V(x) + g |\psi(x)|^2\]

where \(\hbar\) is the reduced Planck’s constant, \(m\) is the mass of the particle, \(V(x)\) is the external potential, and \(g\) is the interaction strength.

Parameters:

  • omega (float) – Angular frequency of the oscillator. Default is 1.0.

  • g (float) – Interaction strength. Default is 1.0.

Usage Examples

Below are examples of how to use the Hamiltonians in qvarnet.

from qvarnet.hamiltonians import HarmonicOscillator

# Initialize the Hamiltonian
hamiltonian = HarmonicOscillator()

# Calculate the energy
local_energies = hamiltonian(positions)

# Supposing that `positions` is distributed following the wavefunction
print(f"Calculated energy: {torch.mean(local_energies)}")
